- `r3s0nant0n3`

Core direction came from real clone-handheld debugging goals:

- make a real terminal shell, not another theme tool
- expose what the device is actually doing
- separate UI intent from backend outcome
- make the workflow shareable for other owners of similar EE/EmuELEC clones
